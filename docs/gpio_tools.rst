Command-line tools
==================

The libgpiod project includes a suite of command-line tools to facilitate GPIO
manipulation from console and shell scripts.

* `gpiodetect <gpio-tools/gpiodetect.html>`_
* `gpioinfo <gpio-tools/gpioinfo.html>`_
* `gpioget <gpio-tools/gpioget.html>`_
* `gpioset <gpio-tools/gpioset.html>`_
* `gpiomon <gpio-tools/gpiomon.html>`_
* `gpionotify <gpio-tools/gpionotify.html>`_
